# 東京8天7夜旅遊規劃

這是一個詳細的東京旅遊規劃網頁，涵蓋了2026年1月20日至1月28日的8天7夜行程。

## 內容

- **基本資訊**：日期、機場、住宿、成員、預算等
- **行程規劃**：每日詳細行程安排
- **實用資訊**：交通、美食、購物、注意事項
- **備選景點**：其他可選的旅遊地點
- **緊急聯絡**：重要電話和聯絡資訊
- **檢查清單**：出發前和每日檢查項目

## 使用方式

直接在瀏覽器中打開 `index.html` 即可查看完整的旅遊規劃。

## GitHub Pages 部署

此網頁已準備好部署到 GitHub Pages：

1. 在 GitHub 上創建一個新的公開倉庫
2. 將 `index.html` 上傳到倉庫
3. 在倉庫的 Settings → Pages 中啟用 GitHub Pages
4. 選擇 main/master 分支作為源
5. 幾分鐘後，網站將在 `https://[你的用戶名].github.io/[倉庫名稱]/` 上線

## 注意事項

- 此規劃為參考性質，請根據實際情況調整
- 部分景點需要提前預約（如迪士尼、TeamLab Planets等）
- 1月天氣較冷，請注意保暖
- 建議提前查看天氣預報，特別是富士山行程
